# Design Tokens

All design decisions are expressed as CSS custom properties. Import `@coe/tokens/css` to use them globally, or import named exports from `@coe/tokens` for JavaScript usage.

## Semantic Colors

| Role | Value | Token |
|---|---|---|
| Primary | `#194588` Navy | `--color-primary` |
| Accent / Link | `#cc4927` Orange | `--color-accent` / `--color-link` |
| Body text | `#0a0a0a` Near Black | `--color-text` |
| Muted text | `#535353` Gray | `--color-text-muted` |
| Faint text | `#888888` | `--color-text-faint` |
| Background | `#ffffff` White | `--color-bg` |
| Surface | `#f8f7f5` Warm off-white | `--color-surface` |
| Border | `#e2e0dc` | `--color-border` |

## Typography

**Heading font:** Museo Sans Display → fallback Nunito, Georgia, sans-serif  
**Body font:** Lato, sans-serif  
**Mono font:** IBM Plex Mono, monospace

| Role | Desktop | Mobile | Line Height | Token |
|---|---|---|---|---|
| H1 | 32px | 28px | 1.3 | `--size-h1` |
| Body | 16px | 16px | 1.8 | `--size-body` |
| Post header | 30px | — | 1.0 | `--size-post-header` |
| Primary nav | 14px | — | — | `--size-nav-primary` |
| Widget heading | 18px | — | — | `--size-widget-heading` |
| Widget body | 14px | — | 1.7 | `--size-widget-body` |
| Button | 20px | — | — | `--size-btn` |
